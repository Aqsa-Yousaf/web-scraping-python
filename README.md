# Faculty Data Scraper

A Python script to scrape faculty information from a university website using BeautifulSoup. Saves the data to a CSV file and downloads profile images locally.

## What it does
Scrapes faculty names and designations from a public webpage, saves data to a CSV file, and downloads faculty images to a local folder (not included in repo).

## Tech used
Python, BeautifulSoup, Requests, Jupyter Notebook

## Files
faculty-web-scraping.ipynb : the main notebook
faculty_data.csv : scraped output
university_info.txt : source URL

## How to run
1. pip install beautifulsoup4 requests
2. Open the notebook in Jupyter
3. Run all cells
